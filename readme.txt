This software is the matlab+mex re-implementation version of the algorithm in 
the following IEEE TCYB15 paper.

Please run the demo code: 
"StereoStrokeCutMain.m" for generating the interactive segmentation with strokes.
"StereoSaliencyCutMain.m" for automatically generating segmentation by saliency maps.
